# HELION — Briefing de Logo

## 1. Sobre a Marca

**Nome:** Helion (pronuncia-se "re-li-on")

**Origem do nome:**
- **Helm** = leme, quem comanda
- **Ion** = energia, partícula carregada
- **Helios** = sol em grego (conexão secundária)

**Significado:** Comando + Energia. Quem está no controle.

---

## 2. O Produto

**O que é:** SaaS/CRM completo para empresas que vendem ao governo brasileiro via licitações públicas.

**Diferencial:** Único sistema que cobre o ciclo completo (captação → disputa → contrato → pagamento) com automação por IA.

**Público:** Empresas brasileiras que participam de licitações (ME, médias empresas, prestadoras de serviço).

## 4. Diretrizes do Logo

### 4.1 Tom

- Futurista mas não frio
- Tech mas não genérico
- Poderoso mas não agressivo
- Sofisticado mas acessível

### 4.2 Estilo

- **Minimalista** — linhas limpas, sem excesso
- **Geométrico** — formas precisas, estruturadas
- **Contemporâneo** — não datado, atemporal

### 4.3 Tipografia

- Sans-serif moderna
- Pode ser lowercase ou title case (NÃO all caps genérico)
- Peso médio a bold
- Personalidade própria (não genérica)

### 4.4 Cores (sugestão inicial, aberto a propostas)

| Cor | Uso |
|-----|-----|
| Azul profundo / Navy | Confiança, tecnologia |
| Cyan / Teal | Inovação, energia |
| Gradientes sutis | Modernidade |
| Branco / Dark mode | Versatilidade |

### 4.5 Ícone

- Deve funcionar sozinho (favicon, app icon)
- Não pode ser: círculo genérico, hexágono básico, engrenagem, gráfico de barras
- Pode explorar: conceito de comando/leme, energia/radiação, estratégia, movimento

## 5. Referências Visuais

### 5.1 Logos de SaaS/Tech que admiro (estilo)

**Linear** (linear.app)
- Minimalista extremo
- Ícone geométrico simples mas único
- Tipografia clean

**Notion** (notion.so)
- Simplicidade icônica
- Funciona em qualquer tamanho
- Personalidade própria

**Vercel** (vercel.com)
- Triângulo como marca forte
- Preto e branco funciona
- Bold e confiante

**Stripe** (stripe.com)
- Wordmark com personalidade
- Gradientes sutis
- Profissional mas moderno

**Loom** (loom.com)
- Ícone que representa o produto (câmera)
- Cores vibrantes mas não infantis
- Memorável
